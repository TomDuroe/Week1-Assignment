Introduction to GitHub

GitHub is a code hosting platform for version control and collaboration. It lets you and others work together on projects from anywhere. This tutorial teaches you GitHub essentials like Branches, Commits, and Pull Requests.

To begin we must firstly understand some key terms which will help you on your journey using GitHub. Please click the links to direct you to some helpful information.

  • Git Commit
  • Git Branch
  • Git Pull Request
  • Git Repository

Step 1, Create an account

To get started you'll need an account. On the GitHub website you'll see a Sign Up button in the top left corner. Click it and follow the simple instructions.

Step 2, Create a Repository

If you havent done so yet, click the GitRepository link above for a some helpful information.

Once you have your GitHub account, you'll need to create a Repository to create a project folder. Follow the steps bellow to get started.

  1. In the upper right corner, next to your avatar, click + and then select New Repository.
  2. In the repository name box, give your repository a name.
  3. In the description box, give your repository a depscription.
  4. Select whether you want your repository to be public or private
  5. Select the Initialize this repository with a README box.
  6. Click the Create Repository box and thats it. You have created your first Repository

Step 3, Create a branch

If you havent done so yet, click the GitBranch link above for a some helpful information.

Once you have your first repository, you'll need to create a branch to be able to work on different versions of your project at the same time. Follow the steps bellow to get started.

  1. Go to your newly made repository
  2. Click the drop down that says Branch Main.
  3. Type a branch name into the Fid or create a branch text box.
  4. Select the Create branch.textbox
  5. Now you have 2 branches, the main branch and the one you have just created.

Step 4, Create a Commit request

If you havent done so yet, click the GitCommit link above for a some helpful information.

Once you have an additional branch for your main branch, you'll need to create some edits as they currently look the same as each other. To do this, follow the steps bellow.

  1. Click the new branch file you have just created
  2. Click the pencil icon in the bottom left which says Edit this file.
  3. In the editor box write some text.
  4. Write a commit message in the text box that describes your changes
  5. Click the Commit changes. button and thats it.

What next?

Congratulations. You now know how to create a project, create additional branches and submit a commit request. The next helpful guide will be on how to use Unix Shell to get the best out of GitHub. Click the UnixShell link to continue your basic guide.
